# WEDID
플레이스토어 링크  https://play.google.com/store/apps/details?id=com.wedid

#### WEDID은 일정 관리 어플리케이션입니다.
#### 일정을 저장하고 주간 캘린더, 월간 캘린더로 일정을 확인할 수 있습니다.
#### 날짜 선택, 스와이프를 통해 간편하게 다음 일정을 확인할 수 있습니다.
#### 하루 종일 일정, 특정 시간 일정을 시간 순으로 볼 수 있습니다.
